# 这是一个什么网站？
这是一个基于nodejs的express框架，用jade模板引擎搭建的小网站。
作者要认真学习数据结构这门深奥的课程，故建立了一个这样的小家，意图来承载零碎的知识笔记。

<br/>
<br/>

# 这个网站对我有什么用？
你可以在这里浏览该网站上上传的笔记文章，从而吸收一些关于计算机数据结构的知识，希望可以对你有所帮助。
